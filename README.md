# 🍏 Sovereign OS | macOS Tahoe (Ultimate Realistic Edition)

> "They can burn the archives, they can filter the network, but they cannot erase an idea."

Sovereign OS (macOS Tahoe Ultimate Edition) is an ultimate interactive web narrative experience built with a pure frontend tech stack (**HTML / Tailwind CSS / Vanilla JS**). It replicates the skuo-morphic glassmorphism UI of a conceptual macOS 26 with **Ultra-Realistic** detail in the browser. You will transform into a hacker, infiltrating from the lock screen to break the global information censorship system codenamed **DIRECTIVE_77** using terminal commands.

---

## ✨ Ultimate Features

### 🔐 Immersive Boot & Login

- **Boot Sequence:** Includes an Apple Logo progress bar boot sequence.
- **Lock Screen:** New Mac lock screen with Gaussian blurred background and realistic avatar (Enter any password and press Enter to enter the system).

### 🖥️ Ultra-Realistic macOS UI/UX

- **Perfect Dock Magnification:** Programmatic dock zoom effect rewritten via JavaScript physics algorithms, perfectly restoring native macOS "ripple" magnification and squeezing effects.
- **Control Center:** Click the icon on the right side of the top menu bar to call out a native glassmorphism control center with Wi-Fi, Bluetooth, and display brightness sliders.
- **Context Menu:** Right-click on empty desktop areas to invoke a native context menu (supporting new terminal, language switching, etc.).
- **Active/Inactive Window States:** Perfectly restores real window hierarchy focus logic; traffic lights automatically grey out when a window loses focus.

### 📊 Data Visualization App Clones

- **TextEdit:** A document reader with a native formatting toolbar.
- **Activity Monitor:** Real-time CPU dynamic load waveform rendering based on SVG, moving beyond static text.

### ⌨️ Enhanced Virtual File System Terminal

- **Bash Parser:** Built-in powerful lightweight Bash command line parser (`admin@sovereign-node ~ %`).
- **Real Command Support:** Supports `ls` (list files), `cat directive_77.txt` (read file), `whoami`, `date`, `clear`, etc.
- **Hacked State (Visual Hijack):** After entering the ultimate command `override`, the system state is rewritten, the dynamic fluid wallpaper turns blood red instantly with a global screen flash, providing intense narrative impact.

### 🌐 Smooth i18n Hot-switching

- Supports Chinese/English dual-language soft refresh, covering global menu bars, windows, document content, and Dock native floating tooltips.
